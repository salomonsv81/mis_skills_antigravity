# Guía Maestra: Skill Architect (Meta-Skill)

## 1. Filosofía Autopoiética
Esta Meta-Skill transforma tu IDE Antigravity en un sistema autopoiético (que se crea a sí mismo). En lugar de escribir manualmente archivos SKILL.md y crear carpetas, delegas esta tarea al **Skill Architect**. Tú defines el **qué** (intención), y el arquitecto define el **cómo** (implementación técnica).

## 2. Capacidades del Architect
* **Análisis de Intención:** Decide si la skill debe ser Global o Workspace.
* **Scaffolding Inteligente:** Crea la estructura de directorios (`scripts/`, `resources/`, `examples/`).
* **Generación de Prompt:** Redacta el SKILL.md inicial con las mejores prácticas.
* **Sanitización:** Asegura nombres en `kebab-case`.

## 3. Estándares que aplica
* **Divulgación Progresiva:** Prompts concisos para ahorrar tokens.
* **Seguridad:** Incluye Safety Guardrails por defecto.
* **Verbosidad Semántica:** Usa verbos de acción en descripciones YAML.
